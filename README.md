# Simple Python Web Server
A Simple Python Web Server Container Image

## Getting Started

### Installation

```bash
docker build -t simple-python-web-server:latest .
```

or

```bash
make build
```

### Usage

1. Place web content at /web/html (via container mount)
2. Run!
